Welcome To IPB Online Banking System

The project simulates online banking system. It allows to register/login, deposit/withdraw money from accounts, add/edit recipients, transfer money between accounts and recipients, view transactions, make appointments. There are two roles user and admin. The admin has there own frontend implemented in Angular2, which communicates with backend through REST services.
